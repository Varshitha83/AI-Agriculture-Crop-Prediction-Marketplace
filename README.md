# 🌾 AI-Driven Agriculture Platform with Crop Prediction and Smart Marketplace

An AI-powered agriculture platform designed to help farmers make better decisions about crop selection and selling strategies.

## 🚀 Features

* Crop prediction using Machine Learning (Random Forest)
* Smart marketplace for direct farmer-to-buyer interaction
* User-friendly web interface
* Data-driven agricultural insights

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Server:** Apache (XAMPP)
* **Database:** MySQL
* **Machine Learning:** Random Forest (Python)

## 📊 Dataset

Includes soil nutrients (N, P, K), pH levels, temperature, and rainfall data.

## 📈 Results

Achieved approximately **92% accuracy** in crop prediction.

## 🔮 Future Improvements

* IoT sensor integration
* Mobile app development
* Multi-language support

## 📌 Conclusion

A farmer-centric platform combining AI prediction with market access to improve productivity and profitability.
